## 🌟 Overview

**E-Mall** is a comprehensive multi-vendor e-commerce platform that connects customers with sellers across various categories including electronics, fashion, home goods, food, and more. Our platform provides a seamless shopping experience with dedicated interfaces for customers, shop owners, and administrators, supporting both physical and digital products.

## 🚀 Quick Start

### Prerequisites

```bash
Node.js >= 14
MongoDB
npm or pnpm
```

### 🔧 Installation

1. **Clone & Install**

```bash
# Clone the repository
git clone https://github.com/your-username/e-mall.git

# Navigate to project
cd e-mall

# Install dependencies
npm install
```

2. **Configure Backend**

```bash
# Navigate to backend
cd backend

# Setup environment
cp .env.example .env

# Install backend dependencies
npm install
```

3. **Start Development Servers**

```bash
# Terminal 1: Start backend
cd backend
npm run dev

# Terminal 2: Start frontend
cd ..
npm run dev
```
